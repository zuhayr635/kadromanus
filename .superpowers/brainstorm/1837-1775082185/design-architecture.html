Değiştirilecek / Eklenmesi Gereken Dosyalar

GÜNCELLENİYOR

server/_core/index.ts

Express app'ten http.createServer() ile HTTP sunucusu oluştur → Socket.io'ya ver → initSocketServer(httpServer) çağır

BÜYÜK GÜNCELLEMEe

server/socket-server.ts

TikTok bağlantı yönetimi ekle. startSession() / stopSession() export et. Gift/like eventlerini yakala, game-engine'i çağır, Socket.io ile yay.

KÜÇÜK GÜNCELLEME

server/routers/broadcaster.ts

createSession → startSession() çağır
endSession → stopSession() çağır

KÜÇÜK GÜNCELLEME

client/src/pages/BroadcasterPanel.tsx

socket.io-client import et, sessionId ile odaya katıl, game:card-opened ve game:state-update dinle

ZATEN HAZIR

client/public/game-screen-websocket.html

Socket.io client kodu mevcut. Sadece sunucu tarafı eventlerini göndermeye başlayınca çalışacak.
